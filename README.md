# Canada Immigration Data Preprocessing (1980–2013)

## Project Description
This project demonstrates the preprocessing and restructuring of the *Canada Immigration by Country (1980–2013)* dataset using Python and Pandas. The primary objective is to transform the raw dataset into a clean, well-structured format suitable for data analysis, visualization, and machine learning applications.

The workflow emphasizes best practices such as inspecting the raw data, and documenting each preprocessing step for clarity and reproducibility.

---

## Dataset Information
- Dataset Name: Canada Immigration by Country
- Time Period: 1980 – 2013
- Level: Country-level immigration data
- Source: Government of Canada (accessed via Google Sheets / Excel file)

The dataset includes immigration counts by country along with metadata such as continent, region, and development status.

---

## Objectives
- Load and inspect the raw dataset
- Display the dataset before preprocessing
- Clean and rename columns for readability
- Remove unnecessary metadata columns
- Restructure the dataset using country names as the index
- Convert year-based columns to numeric values
- Compute total immigration per country
- Produce an analysis-ready DataFrame

---

- Pandas
- Jupyter Notebook
